# How to create a node server:
On a git bash terminal, run this commands: <br />
+ 1º: npm init -y
+ 2º: npm install express
+ 3º: npm install nodemon -D
+ 4º: inside package.json, create script: "dev": "nodemon src/server.js"
+ 5º: npm run dev
